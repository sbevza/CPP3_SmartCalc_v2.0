SmartCalc v2.0 - интеллектуальный калькулятор с расширенными возможностями.

Калькулятор разработан на языке программирования C++, используя объектно-ориентированный подход с использование паттерна MVC.
В дополнение к основным арифметическим операциям, таким как сложение, вычитание, умножение и деление, калькулятор позволяет выполнять вычисления арифметических выражений с учетом приоритетов, а также использовать некоторые математические функции, такие как синус, косинус, логарифм и другие. Кроме того, калькулятор поддерживает использование переменной X и построение графика соответствующей функции.

Дополнительные функции умного калькулятора позволяют расчитать проценты по вкладам и кредитам.

Функции и операторы

Умный калькулятор использует постфиксную польскую нотацию и поддерживает следующие математические операторы и функции:

Арифметические операторы:

- Скобки: "(a + b)"
- Сумма: "a + b"
- Разность: "a - b"
- Умножение: "a * b"
- Деление: "a / b"
- Возведение в степень: "a ^ b"
- Модуль: "a mod b"
- Унарный плюс: "(+a)"
- Унарный минус: "(-a)"

Функции:

- cos(x)
- sin(x)
- tan(x)
- acos(x)
- asin(x)
- atan(x)
- sqrt(x)
- ln(x)
- log(x)

При этом любые функции можно отобразить в виде графика.

Расширенные функции

Кредитный калькулятор позволяет рассчитать ежемесячный платеж, переплату по кредиту и общую сумму выплат. Для этого необходимо ввести следующие входные данные: сумма кредита, срок кредита, процентная ставка и тип кредита (аннуитетный или дифференцированный). После расчета калькулятор выводит ежемесячный платеж, переплату по кредиту и общую сумму выплат.

Депозитный калькулятор позволяет рассчитать начисленные проценты, сумму налога и сумму на вкладе к концу срока. Для этого необходимо ввести следующие входные данные: сумма вклада, срок размещения, процентная ставка, налоговая ставка, пери

одичность выплат, капитализацию процентов, список пополнений и список частичных снятий.

Команды установки и удаления:

- Установка калькулятора: make install
- Удаление калькулятора: make uninstall

Дополнительные команды:

- Проверка стиля кода: make check
- Запуск тестов: make test
- Отчет Gcov: make gcov_report
- Создание архива: make dist
